# Cafe Finder

## Description
Web app untuk mencari café terdekat menggunakan Leaflet + SERP API.

## Features
- Map interaktif dengan user location
- Marker café + popup info
- Search café berdasarkan keyword
- Responsive UI

## Tech Stack
- Frontend: HTML, CSS, JS, Leaflet
- Backend: Node.js, Express, SERP API

## How to Run
1. Clone repo
2. Install backend dependencies: `npm install`
3. Run backend: `node server.js`
4. Open frontend `index.html` or deploy online
